The AI-Based Smart Trolley for Smart Shopping is an innovative IoT-based system designed to automate the traditional shopping and billing process in supermarkets and retail stores. The main aim of this project is to eliminate long checkout queues, reduce manual errors, and enhance customer convenience by providing a seamless and efficient shopping experience. The system uses RFID technology to automatically detect products placed in the cart, while a NodeMCU (ESP8266) microcontroller processes the data and enables real-time communication through a web interface. Whenever a product is added to the cart, the RFID reader scans the tag and retrieves product details such as name and price from a database, which are then displayed on an LCD screen and updated on a web page simultaneously.

The smart trolley allows users to view their selected items and total bill in real time, helping them manage their expenses effectively. It also includes a push-button mechanism to remove items from the cart, ensuring flexibility and accurate billing. The system integrates both hardware components like RFID reader, NodeMCU, LCD display, and software technologies such as Arduino IDE, HTML, CSS, and JavaScript to create a complete automated solution. By reducing dependency on manual barcode scanning and cashiers, the system improves efficiency, minimizes human errors, and speeds up the checkout process.

This project demonstrates the effective use of IoT in modern retail environments by enabling real-time product tracking, automated billing, and remote monitoring through a web interface. It offers several advantages such as reduced waiting time, improved accuracy, better customer satisfaction, and efficient store management. Although there are some challenges like initial setup cost and dependency on internet connectivity, the system provides a scalable and future-ready solution. Future enhancements may include AI-based product recommendations, mobile application integration, online payment options, inventory management, and advanced security features. Overall, the AI-based smart trolley represents a significant step towards smart retail and automated shopping systems.
